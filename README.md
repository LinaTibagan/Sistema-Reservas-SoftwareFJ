# Proyecto Fase 4 - Programación
Sistema de gestión para Software FJ

En este proyecto se está desarrollando un sistema en Python para
administrar clientes, servicios y reservas, trabajando con
Programación Orientada a Objetos y manejo de excepciones.

La idea es que el sistema pueda seguir funcionando incluso cuando
se presenten errores, validando datos y registrando los problemas
en un archivo de logs.

Archivos principales:
- cliente.py → manejo de clientes
- servicio.py → servicios y costos
- reserva.py → creación y control de reservas
- excepciones.py → excepciones personalizadas
- logs.txt → registro de errores y eventos
- main.py → pruebas y ejecución del sistema

Importante:
Antes de comenzar a trabajar ejecutar:
git pull

Y después de realizar cambios:
git add .
git commit -m "descripción"
git push

La idea es mantener el código organizado y que todos podamos
trabajar sobre el mismo proyecto sin problemas.
